# Zenith Maintenance

## Deskripsi Proyek
Zenith Maintenance adalah sebuah situs web yang menampilkan halaman pemeliharaan dengan berbagai fitur interaktif seperti hitungan mundur, efek partikel, dan musik latar. Halaman ini dirancang untuk memberikan informasi kepada pengunjung bahwa situs sedang dalam pemeliharaan dan akan segera kembali online.

## Fitur Utama
- **Hitungan Mundur:** Menunjukkan waktu hingga situs kembali online.
- **Efek Partikel:** Efek partikel yang menarik di latar belakang.
- **Musik Latar:** Musik latar yang diputar otomatis.
- **Efek Teks Scramble:** Efek teks scramble untuk pesan pemeliharaan.
- **Informasi Pengunjung:** Mengambil dan menampilkan informasi pengunjung dari layanan IP info.
- **Tautan Sosial Media:** Tautan ke profil media sosial.

## Instalasi
1. Clone repositori ini:
    ```sh
    git clone https://github.com/Creative-Trees/Zenith-Maintenance.git
    ```
2. Masuk ke direktori proyek:
    ```sh
    cd ZenithMaintenance
    ```

## Struktur Proyek
```
Zenith Maintenance
├── .DS_Store
├── .htaccess
├── assets/
│   ├── .DS_Store
│   ├── css/
│   │   └── style.css
│   ├── js/
│   │   ├── .DS_Store
│   │   ├── app.js
│   │   ├── IP.txt
│   │   ├── music.js
│   │   └── visitor.js
│   └── music/
├── index.html
├── LICENSE
├── readme.md
```
- `.htaccess`: File konfigurasi untuk server web Apache.
- `assets/`: Direktori yang berisi aset seperti CSS, JavaScript, dan musik.
  - `css/style.css`: File CSS untuk styling halaman.
  - `js/app.js`: File JavaScript untuk inisialisasi efek partikel dan teks scramble.
  - `js/music.js`: File JavaScript untuk memutar musik latar.
  - `js/visitor.js`: File JavaScript untuk mengambil dan menampilkan informasi pengunjung.
  - `music/`: Direktori untuk file musik latar.
- `index.html`: File HTML utama yang menampilkan halaman pemeliharaan.
- `LICENSE`: File lisensi proyek.
- `readme.md`: File dokumentasi proyek.
